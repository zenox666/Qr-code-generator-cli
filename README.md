# Qr-code-generator-cli
A simple Node.js CLI application that takes a URL as input and generates a QR code image using Inquirer and qr-image.

---

## Features

- Interactive command-line input using Inquirer
- Generates a QR code in PNG format
- Saves the entered URL to a text file
- Simple and beginner-friendly Node.js project

---

## Technologies Used

-Node.js
-Inquirer
-qr-image
-File System (fs)

---

## Installation

Clone the repository:
```bash
git clone https://github.com/zenox666/Qr-code-generator-cli.git
cd qr-code-generator-cli
```

Install dependencies:
```bash
npm i inquirer qr-image
```

---

## Usage

Run the application:
```bash
node index.js
```
